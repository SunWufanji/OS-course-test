# 项目概览

## 项目名称
**ProcessOS — 进程调度可视化模拟器**

## 项目简介
本系统模拟操作系统的进程管理功能，实现了进程的创建、调度、同步和通信。采用 React 前端 + Spring Boot 后端的前后端分离架构，并集成了 MySQL 数据库持久化。

## 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| **前端** | React + Vite + Axios | React 18 / Vite 5 |
| **后端** | Java + Spring Boot + JPA | Java 17 / Spring Boot 3.5 |
| **数据库** | MySQL + Hibernate | MySQL 8.0 |
| **构建工具** | Maven + npm | Maven 3.9 |

## 核心功能
1. **进程管理** — 创建、删除、挂起、恢复进程
2. **调度算法** — FCFS、SJF、SRTN、RR、优先级、抢占式优先级、MFQ（多级反馈队列）
3. **进程树** — 父子进程关系，fork/级联终止
4. **同步互斥** — 生产者消费者、哲学家就餐、读者写者
5. **数据库持久化** — 执行轨迹、性能指标、实验场景
6. **桌面系统** — 仿 Windows WebOS 桌面 + 任务管理器
7. **硬件资源模拟** — 多核 CPU、内存池、IO 设备

## 项目特点
- 从单一调度算法演示升级为完整的操作系统模拟器
- 双模架构：沙盒模式（实时模拟）+ 实验室模式（离线算法演示）
- 毛玻璃（Glassmorphism）视觉风格
