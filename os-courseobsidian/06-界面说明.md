# 界面说明

## 一、桌面系统（主界面）

### 布局
- **背景**：深色极光渐变壁纸，科技感十足
- **桌面图标**：网格排列，包含系统应用和用户可启动的应用
- **任务栏**：底部 48px 高度，毛玻璃半透明效果

### 桌面图标
| 图标 | 名称 | 功能 |
|------|------|------|
| ⚙️ | 任务管理器 | 查看进程列表和系统资源监控 |
| 🧪 | 内核算法实验室 | 调度算法演示（旧版全部功能） |
| 🎮 | CS:GO | 启动游戏进程（模拟） |
| 🌐 | Chrome | 启动浏览器进程（模拟） |
| 💻 | VSCode | 启动编辑器进程（模拟） |
| 📝 | 记事本 | 启动轻量进程（模拟） |
| ... | 共 15 种应用 | 各有不同的 CPU/内存/IO 需求 |

### 交互
- **双击图标**：启动应用，在桌面弹出窗口
- **Hover 图标**：放大 + 蓝色发光效果

### 任务栏
- **左侧**：开始菜单按钮
- **中间**：运行中的应用标签（可点击切换窗口）
- **右侧**：CPU 使用率条 + 内存使用率条 + 实时时钟

## 二、窗口系统

### 窗口组件
- **标题栏**：可拖拽移动，双击最大化/还原
- **控制按钮**：最小化（─）、最大化（☐）、关闭（✕）
- **视觉效果**：毛玻璃半透明背景 + 圆角 12px + 投影

### 窗口行为
- 点击窗口任意区域 → 激活到最前
- 最小化 → 缩到任务栏，点击任务栏标签恢复
- 最大化 → 全屏（不含任务栏），再次点击还原

## 三、内核算法实验室

### 布局
左侧配置面板 + 右侧结果展示

### 左侧面板
- **调度算法选择**：FCFS / SJF / RR / 优先级（按钮组）
- **时间片设置**：RR 算法时可调整
- **进程列表**：显示已配置的进程，支持删除
- **添加进程**：名称 / 执行时间 / 优先级 / 到达时间
- **操作按钮**：开始模拟 / 加载演示

### 右侧展示
- **甘特图**：彩色时间轴 + 红色扫描线动画
- **统计信息**：平均周转 / 平均等待 / CPU 利用率 / 已完成数
- **进程详情表**：每个进程的到达/执行/完成/周转/等待时间

### 附加功能
- **进程树**：树形展示父子进程关系
- **同步互斥演示**：生产者消费者 / 哲学家就餐 / 读者写者
- **历史记录**：查看/删除/重新创建历史模拟
- **实验场景**：预设轻载/中载/重载等场景
- **数据库保存**：手动保存当前模拟结果

## 四、任务管理器

### 进程列表
| 列 | 说明 |
|----|------|
| 图标 | 应用 emoji 图标 |
| 名称 | 进程名称 |
| PID | 进程 ID |
| 状态 | 运行中（绿色）/ 已挂起（黄色） |
| CPU | 当前 CPU 占用率 |
| 内存 | 内存占用 (MB) |
| 磁盘读 | 磁盘读取速度 |
| 网络 | 网络速度 |
| 操作 | 挂起/恢复 + 结束按钮 |

### 性能监控
- **CPU**：8 核心独立显示使用率柱状图
- **内存**：使用率进度条 + 分配详情
- **磁盘**：读取/写入速度
- **网络**：传输速度

### 交互
- 点击列标题排序
- 运行中进程旁有绿色闪烁点
