# 快速开始

## 环境要求

| 工具 | 版本 | 说明 |
|------|------|------|
| JDK | 17+ | Java 运行环境 |
| Maven | 3.6+ | 后端构建工具 |
| Node.js | 18+ | 前端运行环境 |
| npm | 9+ | 前端包管理器 |
| MySQL | 8.0 | 数据库 |

## 1. 数据库初始化

```sql
-- 登录 MySQL 执行建表脚本
mysql -u root -p < backend/src/main/resources/schema.sql
```

或者手动执行：
```sql
CREATE DATABASE process_os;
USE process_os;
-- 然后执行 schema.sql 中的内容
```

默认账号密码：`root / 123456`（可在 `application.properties` 中修改）

## 2. 启动后端

```bash
cd backend

# 首次启动（下载依赖）
mvn clean install

# 启动服务
mvn spring-boot:run
```

后端运行在 `http://localhost:9090`

## 3. 启动前端

```bash
cd frontend

# 首次启动（安装依赖）
npm install

# 启动开发服务器
npm run dev
```

前端运行在 `http://localhost:3000`

## 4. 访问系统

打开浏览器访问 `http://localhost:3000`

- 桌面上双击图标启动应用
- 双击"内核算法实验室"打开调度算法演示
- 双击"任务管理器"查看进程和资源监控

## API 接口速查

### 系统管理（沙盒模式）
```bash
GET  /api/system/status          # 系统状态
GET  /api/system/apps            # 可用应用列表
POST /api/process/launch         # 启动应用 {"appName":"CS:GO"}
POST /api/process/{pid}/terminate # 结束进程
POST /api/process/{pid}/suspend  # 挂起进程
POST /api/process/{pid}/resume   # 恢复进程
GET  /api/process/sandbox        # 沙盒进程列表
POST /api/system/reset           # 重置沙盒
```

### 调度算法
```bash
GET  /api/processes              # 获取所有进程
POST /api/processes              # 创建进程
POST /api/tick                   # 推进时间
POST /api/scheduler              # 切换算法 {"algo":"FCFS"}
POST /api/quantum                # 设置时间片 {"quantum":2}
POST /api/demo                   # 加载演示数据
POST /api/reset                  # 重置系统
```

### 实验室
```bash
POST /api/lab/simulate           # 运行调度模拟
```

### 数据库
```bash
POST /api/save                   # 保存结果
GET  /api/history                # 历史记录
GET  /api/scenarios              # 实验场景
POST /api/scenarios/{id}/load    # 加载场景
```
