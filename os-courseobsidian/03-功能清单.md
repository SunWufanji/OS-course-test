# 功能清单

## 一、进程管理

| 功能 | 说明 | 状态 |
|------|------|------|
| 创建进程 | 设置名称、执行时间、优先级、到达时间 | ✅ |
| 删除进程 | 根据 PID 删除进程 | ✅ |
| 挂起进程 | 将就绪/运行进程转为阻塞状态 | ✅ |
| 恢复进程 | 将阻塞进程转为就绪状态 | ✅ |
| 进程状态流转 | 新建→就绪→运行→阻塞→结束 | ✅ |

## 二、调度算法

| 算法 | 英文名 | 特点 | 状态 |
|------|--------|------|------|
| 先来先服务 | FCFS | 按到达顺序执行 | ✅ |
| 短作业优先 | SJF | 选执行时间最短的 | ✅ |
| 最短剩余时间优先 | SRTN | SJF 的抢占式版本 | ✅ |
| 时间片轮转 | RR | 每个进程运行固定时间片 | ✅ |
| 非抢占优先级 | Priority | 选优先级最高的 | ✅ |
| 抢占式优先级 | PreemptivePriority | 高优先级可抢占 | ✅ |
| 多级反馈队列 | MFQ | 多级队列 + 时间片递增 | ✅ |

## 三、进程树

| 功能 | 说明 | 状态 |
|------|------|------|
| 查看进程树 | 树形展示父子进程关系 | ✅ |
| fork 子进程 | 从选中进程派生子进程 | ✅ |
| 级联终止 | 终止进程及其所有子进程 | ✅ |

## 四、同步互斥

| 功能 | 说明 | 状态 |
|------|------|------|
| 生产者消费者 | 信号量 + 互斥锁实现 | ✅ |
| 哲学家就餐 | 死锁避免演示 | ✅ |
| 读者写者 | 读写锁实现 | ✅ |

## 五、数据库持久化

| 功能 | 说明 | 状态 |
|------|------|------|
| 保存模拟结果 | 执行轨迹 + 性能指标写入 MySQL | ✅ |
| 历史记录 | 查看/删除/重新创建历史模拟 | ✅ |
| 实验场景 | 预设轻载/中载/重载/CPU密集/IO密集场景 | ✅ |

## 六、桌面系统（WebOS）

| 功能 | 说明 | 状态 |
|------|------|------|
| 桌面图标 | 双击启动应用 | ✅ |
| 任务栏 | 显示运行中应用 + 系统托盘 | ✅ |
| 窗口系统 | 拖拽/最小化/最大化/关闭 | ✅ |
| 任务管理器 | 进程列表 + CPU/内存监控 | ✅ |

## 七、硬件资源模拟

|------|------|------|
