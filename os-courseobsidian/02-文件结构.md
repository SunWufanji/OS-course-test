# 文件结构

## 项目根目录

```
OS-course-test/
├── README.md                    # 项目说明（中文）
├── CLAUDE.md                    # Claude Code 开发指南
├── backend/                     # Spring Boot 后端
├── frontend/                    # React 前端
└── os-courseobsidian/           # 课设文档（Obsidian 笔记）
```

## 后端结构 (backend/)

```
backend/
├── pom.xml                      # Maven 依赖配置
└── src/main/
    ├── java/com/processos/
    │   ├── ProcessOsApplication.java      # Spring Boot 主类 + 定时任务
    │   │
    │   ├── model/                         # 数据模型层
    │   │   ├── ProcessControlBlock.java   # 进程控制块 PCB（支持沙盒模式）
    │   │   ├── ProcessState.java          # 进程状态枚举
    │   │   ├── SimulatedApp.java          # 模拟应用列表（15种应用）
    │   │   ├── ExecutionLog.java          # 执行轨迹表（MySQL）
    │   │   ├── ScenarioConfig.java        # 实验场景配置表（MySQL）
    │   │   └── PerformanceMetrics.java    # 性能指标表（MySQL）
    │   │
    │   ├── scheduler/                     # 调度算法层
    │   │   ├── Scheduler.java             # 调度器接口
    │   │   ├── FCFSScheduler.java         # 先来先服务
    │   │   ├── SJFScheduler.java          # 短作业优先
    │   │   ├── SRTNScheduler.java         # 最短剩余时间优先
    │   │   ├── RoundRobinScheduler.java   # 时间片轮转
    │   │   ├── PriorityScheduler.java     # 非抢占优先级
    │   │   ├── PreemptivePriorityScheduler.java  # 抢占式优先级
    │   │   └── MFQScheduler.java          # 多级反馈队列
    │   │
    │   ├── hardware/                      # 硬件资源管理（沙盒模式）
    │   │   ├── HardwarePool.java          # CPU/内存/IO 资源池
    │   │   └── MemoryBlock.java           # 内存块（首次适应算法）
    │   │
    │   ├── process/                       # 进程管理（沙盒模式）
    │   │   └── ProcessManager.java        # 沙盒进程管理器
    │   │
    │   ├── sync/                          # 同步机制层
    │   │   ├── Semaphore.java             # 信号量
    │   │   ├── Mutex.java                 # 互斥锁
    │   │   └── MessageQueue.java          # 消息队列
    │   │
    │   ├── service/                       # 业务逻辑层
    │   │   ├── ProcessService.java        # 核心调度服务
    │   │   └── SyncDemoService.java       # 同步互斥演示服务
    │   │
    │   ├── controller/                    # REST API 层
    │   │   ├── ProcessController.java     # 调度算法 API
    │   │   ├── SystemController.java      # 沙盒系统 API
    │   │   └── LabController.java         # 实验室模拟 API
    │   │
    │   └── repository/                    # 数据访问层
    │       ├── ExecutionLogRepository.java
    │       ├── ScenarioConfigRepository.java
    │       └── PerformanceMetricsRepository.java
    │
    └── resources/
        ├── application.properties         # 应用配置（MySQL + 端口 9090）
        └── schema.sql                     # 数据库建表 + 默认场景
```

## 前端结构 (frontend/)

```
frontend/
├── package.json                   # npm 依赖
├── vite.config.js                 # Vite 构建 + API 代理
├── index.html                     # 入口 HTML
└── src/
    ├── main.jsx                   # React 入口
    ├── App.jsx                    # 桌面系统入口（WebOS）
    ├── index.css                  # 全局样式（毛玻璃 + 桌面 + 旧版）
    │
    └── components/
        ├── Desktop/               # 桌面系统组件
        │   ├── Desktop.jsx        # 桌面主组件
        │   ├── DesktopIcon.jsx    # 桌面图标
        │   ├── TaskBar.jsx        # 任务栏
        │   └── Window.jsx         # 窗口系统
        │
        ├── TaskManager/           # 任务管理器
        │   └── TaskManager.jsx    # 进程列表 + 性能监控
        │
        ├── SchedulerLab/          # 内核算法实验室（旧版全部功能）
        │   └── SchedulerLab.jsx   # 调度算法演示 + 甘特图
        │
        ├── KernelLab/             # 实验室组件（独立版）
        │   └── KernelLab.jsx      # 简化版实验室
        │
        └── MFQPanel.jsx           # 多级反馈队列面板
```
